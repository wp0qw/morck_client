<h4 align="center">Morck是一款同时支持 sspanel、v2borad 的客户端。</h4>

## Introduction

采用clash核心作为主Agent核心。

支持 v2borad面板、sspanel面板及各种sspanel衍生主题。


## Supported

|前端              |v2ray              |trojan           |shadowsocks           |shadowsocksR           |
|------------------|------------------|------------------|------------------|------------------|
|sspanel-uim	   |√                 |暂不支持                 |暂不支持                 |√                 |
|v2board	   |√                 |√                 |√                 |暂不支持                 |

## Function

|功能              |是否支持              |
|------------------|------------------|
|登录                 |√ 
|注册(跳转)                 |√
|显示到期时间                 |√
|显示流量情况                 |√
|套餐续费(跳转)                 |√
|官网入口                 |√
|节点刷新                 |√
|国旗(252个国家/地区)                 |√
|节点延迟(tcping)                 |√
|版本更新                 |√
|智能代理                 |√
|全局代理(tun)                 |√
